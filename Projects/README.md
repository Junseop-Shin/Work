## 새 프로젝트 추가 방법

```bash
# 1. Projects/ 하위에 폴더 생성 후 개별 레포로 초기화
cd ~/Documents/Work/Projects
git clone https://github.com/YOUR_ID/YOUR_REPO

# 2. 이 README.md 테이블에 행 추가
# | 프로젝트명 | 설명 | [링크](GitHub URL) | 상태 |
```

> `Projects/*/`는 Work 레포의 `.gitignore`에 등록되어 있어
> 각 프로젝트의 커밋은 해당 레포에만 반영됩니다.
